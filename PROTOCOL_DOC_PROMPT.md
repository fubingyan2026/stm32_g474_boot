# 嵌入式通信协议文档生成提示词

> 将以下提示词提供给 AI，可生成风格统一、结构清晰的协议规范文档。

---

## 使用方式

```
请根据以下规范，为 [项目名称] 的 [协议名称] 生成一份通信协议规范文档。

[粘贴下方"文档规范"部分]
```

---

## 文档规范

### 1. 头部信息

使用表格呈现元数据：

| 项目 | 信息 |
|------|------|
| **文档版本** | V[主版本].[次版本].[修订] |
| **作者** | [作者名] |
| **创建日期** | YYYY-MM-DD |
| **最后更新** | YYYY-MM-DD |
| **目标 MCU** | [芯片型号] (内核, Flash/RAM) |
| **传输层** | [总线类型] ([标准]) |

### 2. 目录结构

每个章节独立，编号清晰：

```
1. 系统架构          — 一句话概述 + 设计原则表
2. [硬件/存储]布局    — 地址表 + 结构体字段表
3. 通信链路层        — 标识符分配 + 帧格式协商 + 通用结构
4. 协议帧定义        — 命令字一览 + 逐帧详细定义（含字节偏移表）
5. 升级/业务流程      — 时序图(保留) + 断点续传 + 边界处理
6. 超时与错误处理    — 超时参数表 + 状态机表 + 错误恢复表
7. 启动/初始化决策   — 条件判断表
附录               — 帧传输示例(表格) + 关键常数表
```

### 3. 表格规范

**字节偏移表**（帧格式定义的标准写法）：

```
| Byte 0 | Byte 1 | Byte 2 | Byte 3 | ... | Byte N |
|--------|--------|--------|--------|-----|--------|
| [字段1] | [字段2] (MSB) | | | (LSB) | [字段N] |
```

**字段说明表**（紧接字节偏移表）：

```
| 字段 | 偏移 | 长度 | 说明 |
|------|------|------|------|
| [名称] | Byte X | N | [描述]（[类型], [端序]） |
```

**状态行为表**（状态机用表格代替流程图）：

```
| 状态 | 接受的命令 | 处理 | 响应 |
|------|-----------|------|------|
| [状态名] | [命令列表] | [逻辑描述] | ACK / NACK |
```

**状态转移矩阵**：

```
| 从 → 到 | STATE_A | STATE_B | STATE_C |
|---------|:---:|:---:|:---:|
| **STATE_A** | — | [触发条件] | |
| **STATE_B** | [触发条件] | — | [触发条件] |
```

**超时参数表**：

```
| 参数 | 值 | 说明 |
|------|-----|------|
| [名称] | [数值+单位] | [触发条件与行为] |
```

### 4. 帧定义规范

每种帧包含三个部分：
1. **概述**：一句话说明帧的用途和特点
2. **字节偏移表**：精确到每个 byte 的内容
3. **字段说明表**：各字段的类型、长度、端序、取值范围

命令字统一用一览表总览，后续各帧再展开：

```
| 命令字 | 宏定义 | 方向 | 说明 |
|--------|--------|------|------|
| `0x01` | `CMD_XXX` | H→N | [功能描述] |
```

### 5. 时序图

保留 mermaid `sequenceDiagram`，禁止使用 `flowchart`、`stateDiagram` 等——这些用表格替代更清晰。

时序图只标注关键交互点，不标注错误分支（错误分支在"超时与错误处理"章节用表格说明）。

### 6. 附录规范

**帧传输示例**使用表格而非 raw text：

```
| 帧序号 | 格式 | 说明 |
|--------|------|------|
| 0 ~ N | `[0x03] [seq] [D bytes payload]` | N+1 帧 DATA, 累计 X 字节 |
| N+1 | `[0x08] [seq] [cs_H] [cs_L] [R bytes remaining] [P bytes padding]` | DATA_END |
```

**关键常数表**（统一收尾）：

```
| 常量 | 值 | 定义位置 |
|------|-----|----------|
| [常量名] | [值] | [源文件/宏名] |
```

### 7. 禁止事项

- **禁止** mermaid 流程图、状态图（用表格替代）
- **禁止** raw text 帧格式描述（用字节偏移表替代）
- **禁止** 在帧示例中使用裸十六进制字符串（用表格行替代）
- **禁止** 超时/错误处理散落在各章节（统一到第 6 章）
- **禁止** 超过 6 列的字节偏移表（长帧可分行或省略尾部填充列）

### 8. 命名约定

| 语境 | 格式 | 示例 |
|------|------|------|
| 字节值 | 反引号 + `0x` 前缀 | `` `0x01` `` |
| 宏定义 | `` `UPPER_SNAKE_CASE` `` | `` `BOOT_CMD_START` `` |
| 字段名 | `` `snake_case` `` | `` `fw_total_size` `` |
| 地址 | 反引号 | `` `0x08000000` `` |
| 数据类型 | 反引号 | `` `uint32_t` `` |
| 枚举值 | `` `UPPER_SNAKE_CASE` `` | `` `BLOCK_CHECKSUM` `` |
